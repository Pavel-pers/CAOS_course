# Разворот односвязного списка

Рассмотрим односвязный список со следующей структурой узла:

```cpp
struct ListNode {
    int value;
    ListNode* next;
};
```

Ваша задача – реализовать функцию `Reverse`, которая принимает указатель на голову списка и возвращает
указатель на голову развернутого списка.

Требования к реализации:

- Время работы: `O(n)`, где n – количество элементов в списке.
- Отсутствие динамических аллокаций и `O(1)` потребляемой памяти – разворот должен происходить in-place.
