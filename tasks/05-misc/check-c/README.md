# Check-C

Напишите программу, принимающую первым аргументом строку и проверяющую, что эта строка является "полноценной" с точки зрения языка си.
Строка является "полноценной" если она компилируется как тело функции без аргументов и возвращаемого значения и не зависит от переменных/функций (символов) не объявленных в этой же строке.

В случае если строка валидная необходимо вывести Valid, иначе Invalid.
Программа должна завершиться с exit code 0.
Гарантируется, что в строке нет символов `{` и `}`.


## Примеры

Ввод:

```c
int x = 0; x + 3;
```

Ответ:

```
Valid
```

---

Ввод:

```c
1 + 2 +;
```

Ответ:

```
Invalid
```

---

Ввод:

```c
y + 4;
```

Ответ:

```
Invalid
```
