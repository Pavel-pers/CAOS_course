# Решение

В программе-родителе создадим 2 процесса-ребенка.

Один из детей с помощью библиотечной функции [exec*](https://man7.org/linux/man-pages/man3/exec.3.html) будет исполнять переданную команду.
Второй процесс, с помощью функции [sleep](https://man7.org/linux/man-pages/man3/sleep.3.html) будет ожидать время, переданное в утилиту, и после этого завершать свою работу.

Родитель, дождавшись одного ребенка, с помощью системного вызова [kill](https://man7.org/linux/man-pages/man2/kill.2.html) закончит исполнение более долгого ребенка. В качестве сигнала используйте `SIGTERM`

Далее в курсы мы изучим инструменты, с помощью которых можно будет написать эту утилиту более простым способом.
