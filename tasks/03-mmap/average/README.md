### Statement

Программе в аргументе командной строки задается имя непустого бинарного файла,
содержащего массив вещественных чисел двойной точности (`double`).

На стандартный поток вывода напечатайте в шестнадцатеричном виде
среднее арифметическое чисел в файле.

Для печати результата в шестнадцатиричном виде можете использовать
[`std::hexfloat`](https://en.cppreference.com/w/cpp/io/manip/fixed)
из `<iomanip>`, либо спецификатор формата `%a` у `printf`.

Используйте `mmap` для обработки входного файла.
Предполагайте, что входной файл целиком помещается в память.
