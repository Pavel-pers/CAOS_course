# Simple allocator

В этой задаче вам предстоит реализовать примитивный аллокатор. Реализуйте две функции:

- `Allocate16` – аллоцирует блок памяти размером 16 байт и возвращает указатель на его начало
- `Deallocate16` – деаллоцирует блок памяти, аллоцированный ранее функцией `Allocate16`

В случае, если `mmap` вернул ошибку и в аллокаторе нет свободных блоков памяти,
возвращайте `nullptr`. Гарантируется, что переданный в `Deallocate16` указатель был получен
из вызова функции `Allocate16`.

## Реализация

Предлагается реализовать аллокатор с использованием free list – при деаллокации
блок памяти добавляется в односвязный список. При аллокации блока сначала проверяется
free list, и, если он пуст, аллоцируется новая страница с помощью `mmap`. В данном подходе
аллокатор не умеет освобождать страницы. Ваше решение будет собираться
[без стандартной библиотеки](../../../docs/nostd.md).
